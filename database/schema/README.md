# PostgreSQL Schema

This folder contains the PostgreSQL scripts used to create the MedLogs database schema.

Run the files in this order:

1. `01_create_database_postgresql.sql`
2. `02_medico_legal_schema_postgresql.sql`
3. `../seeds/03_sample_master_data_postgresql.sql`

Do not store real patient, medico-legal case, laboratory, or court data in this repository.
